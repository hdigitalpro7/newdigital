File Specifications & Artwork Guidelines

Follow these guidelines to ensure your artwork is print-ready and meets all quality standards.

File Specifications

  • ✔️ Recommended Safe Zone: 3mm inside from all edges
  • ✔️ Accepted File Types: PDF or JPEG
  • ✔️ Resolution: 300dpi for images, 600dpi for text saved as image

Essentials

  • ✔️ Colours must be CMYK (no RGB or other profiles)
  • ✔️ All fonts must be embedded or converted to curves/outlines
  • ✔️ Borders & text must stay within safe zone
  • ✔️ Artwork must match order specifications
  • ✔️ Logos should be supplied in vector format where possible

File Dimensions

  • ✔️ File must match the final print size
  • ✔️ Example: If printing A4, file must be set to A4
  • ✔️ Example: If printing A5, file must be set to A5
  • ✔️ Always create the file in the exact required dimensions

General Instructions

  • ✔️ Artwork must include the required bleed
  • ✔️ All transparency effects must be flattened
  • ✔️ Overprint settings must be correct for the desired result
  • ✔️ Remove all colour profiles from the file
  • ✔️ Ensure images are at the correct resolution
  • ✔️ Remove registration marks, dielines, and other non-printing elements
  • ✔️ Place crop marks outside the bleed area if needed
